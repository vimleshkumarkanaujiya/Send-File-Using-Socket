# Send File Using Socket (Linux System)

## Steps to Run :
In root directory - 
```
cd client
```
Complie using GNU C++ compiler
```
g++ -o client client.cpp
```
Execute
```
./client
```
Let <b>client</b> connection alive.

Go to root directory - 
```
cd server
```
Complie using GNU C++ compiler
```
g++ -o server server.cpp
```
Execute
```
./server
```
All works!
